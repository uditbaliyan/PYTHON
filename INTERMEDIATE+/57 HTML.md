


| Tag             | Description                                 | Example Code                                        |
|-----------------|---------------------------------------------|----------------------------------------------------|
| `<!DOCTYPE>`    | Specifies the HTML version and type         | `<!DOCTYPE html>`                                  |
| `<html>`        | Root element of an HTML page                | `<html>...</html>`                                 |
| `<head>`        | Contains meta-information about the document | `<head>...</head>`                                 |
| `<title>`       | Sets the title of the HTML document         | `<title>Page Title</title>`                       |
| `<meta>`        | Provides meta-information about the document | `<meta charset="UTF-8">`                          |
| `<link>`        | Links external resources like stylesheets   | `<link rel="stylesheet" href="styles.css">`        |
| `<script>`      | Embeds or references scripts (JavaScript)   | `<script src="script.js"></script>`               |
| `<body>`        | Contains the content of the document         | `<body>...</body>`                                 |
| `<h1> to <h6>`  | Define headings of different levels         | `<h1>Heading 1</h1>`                              |
| `<p>`           | Defines a paragraph                         | `<p>This is a paragraph.</p>`                     |
| `<a>`           | Creates a hyperlink                          | `<a href="https://www.example.com">Link</a>`      |
| `<img>`         | Embeds an image                              | `<img src="image.jpg" alt="Description">`         |
| `<ul>`          | Creates an unordered list                    | `<ul><li>Item 1</li><li>Item 2</li></ul>`         |
| `<ol>`          | Creates an ordered list                      | `<ol><li>Item 1</li><li>Item 2</li></ol>`         |
| `<li>`          | Defines a list item                          | `<li>Item 1</li>`                                 |
| `<div>`         | Defines a division or section                | `<div>Content goes here</div>`                    |
| `<span>`        | Defines a span of text within a block-level element | `<span style="color: red;">Red text</span>` |
| `<form>`        | Creates a form                               | `<form action="/submit" method="post">...</form>` |
| `<input>`       | Defines an input field                       | `<input type="text" name="username">`             |
| `<button>`      | Creates a button                             | `<button type="button">Click me</button>`        |
| `<select>`      | Creates a dropdown list                      | `<select><option value="1">Option 1</option>...</select>` |
| `<textarea>`    | Defines a multiline text input               | `<textarea rows="4" cols="50"></textarea>`        |
